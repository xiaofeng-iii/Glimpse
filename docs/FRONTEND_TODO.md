## 设计规范建议

### 颜色方案

| 元素 | 浅色主题 | 深色主题 |
|------|----------|----------|
| 主背景 | `#FFFFFF` | `#1E1E1E` |
| 列表项背景 | `#F5F5F5` | `#2D2D2D` |
| OCR 徽章 | `#E3F2FD` (蓝) | `#1565C0` |
| 语义徽章 | `#E8F5E9` (绿) | `#2E7D32` |
| 选中项 | `#BBDEFB` | `#0D47A1` |
| 文字主色 | `#212121` | `#EEEEEE` |
| 文字次色 | `#757575` | `#9E9E9E` |

### 字体建议

- 主字体: 系统默认无衬线字体（`MS Shell Dlg 2` / `PingFang SC`）
- 列表项字号: 13px
- 详情面板字号: 12px
- 状态栏字号: 11px

### 间距规范

- 控件间距: 8px
- 内边距: 12px
- 列表项高度: 48px（含缩略图）/ 32px（纯文本）
